Invoice Application

Welcome to the Invoice Application. This program was developed for one single reason, which was to learn how to use the print function in VB. This program has turned out to be a big success and I have programmed it so that anybody can use it. This help file was created not to show you how to use this program step by step, but to debug your errors.

If you find that you are trying to print and cannot then follow these instructions:

  1. Have you entered in your ABN (Australian Business Number)? If you haven't then look at the first number in the menu and click it, enter your ABN and click the Done button.
  2. If you are still getting that error, then enter the business address or name by clicking the second button and entering your data.
  3. If you are still getting that error, then enter the person's name in the field at the bottom and along with that enter some fields in the 'Form Entry Page'.
  4. If you are still getting an error, you must be thick.. go back to step 1 and follow through

Anyway, hopefully you will understand how to use this program ALL by YOURSELF!
